# Documentation Openbeehive

**Openbeehive** est un outil de tenue de registres apicoles fonctionnant en mode
hors ligne d'abord — un service hébergé gratuit **et** un logiciel auto-hébergeable,
aussi bien pour les apiculteurs amateurs que professionnels. L'application stocke
tout localement et se synchronise avec le serveur, de sorte qu'elle fonctionne au
rucher avec ou sans signal.

- 🐝 **Utiliser l'application :** [app.openbeehive.org](https://app.openbeehive.org)
- 🌐 **Site du projet :** [openbeehive.org](https://openbeehive.org)
- 💻 **Code source :** [github.com/johnnycube/openbeehive-app](https://github.com/johnnycube/openbeehive-app)
