# Openbeehive-Dokumentation

**Openbeehive** ist eine Offline-First-Lösung zur Dokumentation der
Imkerei — ein kostenloser gehosteter Dienst **und** selbst hostbare Software,
für Hobby- und Berufsimker gleichermaßen. Die App speichert alles lokal und
synchronisiert mit dem Server, sodass sie am Bienenstand mit oder ohne Signal
funktioniert.

- 🐝 **App nutzen:** [app.openbeehive.org](https://app.openbeehive.org)
- 🌐 **Projekt-Website:** [openbeehive.org](https://openbeehive.org)
- 💻 **Quellcode:** [github.com/johnnycube/openbeehive-app](https://github.com/johnnycube/openbeehive-app)
