# Glossar

Die Imkerei hat ihr ganz eigenes Vokabular, und Openbeehive fügt noch ein paar
eigene Begriffe hinzu. Diese Seite versammelt die Wörter, die dir am häufigsten
begegnen werden, sowohl am Stock als auch in der App, mit kurzen, praktischen
Definitionen.

Nutze die Suchfunktion deines Browsers, um direkt zu einem Begriff zu springen,
oder überfliege die Seite einfach. Wo ein Begriff eine eigene Seite hat,
verlinken wir darauf.

:::tip
Neu in der Imkerei? Beginne mit [dem Bienenvolk](/beekeeping/the-colony) und
[den ersten Schritten](/beekeeping/getting-started) und komm dann hierher zurück,
wann immer dich ein Wort stolpern lässt.
:::

## Imkerbegriffe

| Begriff | Bedeutung |
| --- | --- |
| **Bienenstand (Apiary)** | Ein Standort, an dem ein oder mehrere Bienenstöcke gehalten werden. In Openbeehive steht der Bienenstand an der Spitze der Hierarchie und ist die Einheit, über die geteilt wird. Siehe [Bienenstände](/using-the-app/apiaries). |
| **Bienenabstand (Bee space)** | Der etwa 6-9 mm große Spalt, den Bienen als Durchgang frei lassen. Kleinere Räume werden mit Propolis gefüllt, größere mit Wabenbau. Der richtige Bienenabstand hält die Rähmchen beweglich. |
| **Brut** | Der heranwachsende Nachwuchs: Eier, Larven und Puppen. Ein gesundes „Brutbild“ ist geschlossen und weist wenige Lücken auf. |
| **Brutraum** | Die untere Zarge (oder Zargen), in der die Königin legt und Brut aufgezogen wird, im Gegensatz zu den darüber liegenden Honigräumen. |
| **Wildbau (Burr comb)** | Wildwabenbau in Spalten oder zwischen Zargen, außerhalb der Rähmchen. |
| **Verdeckelte Brut** | Puppen, die unter Wachsdeckeln versiegelt sind. Arbeiterinnendeckel sind flach und matt, Drohnendeckel gewölbt. |
| **Zelle** | Ein einzelnes sechseckiges Fach der Wabe. Besondere Zellen sind Weiselzellen, Drohnenzellen und Spielnäpfchen. |
| **Wabe** | Die Wachsstruktur aus sechseckigen Zellen, die zum Lagern von Honig und Pollen sowie zur Aufzucht der Brut dient. |
| **Ausgebaute Wabe** | Mittelwand, die die Bienen zu vollständigen, nutzbaren Zellen ausgebaut haben. |
| **Drohn** | Eine männliche Biene. Drohnen sammeln nicht und haben keinen Stachel; ihre Aufgabe ist die Begattung von Königinnen. |
| **Drohnenbrütige Königin** | Eine versagende oder unbegattete Königin, die nur unbefruchtete (Drohnen-)Eier legt - ein Zeichen, dass sie ersetzt werden muss. |
| **Mittelwand (Foundation)** | Eine Platte aus Bienenwachs oder Kunststoff, oft mit einem Zellmuster geprägt, die den Bienen eine Grundlage zum Wabenbau gibt. |
| **Tracht (Forage)** | Der Nektar, Pollen, das Wasser und Propolis, die Bienen sammeln; auch der Vorgang des Sammelns selbst. |
| **Rähmchen** | Ein herausnehmbarer Holz- oder Kunststoffrahmen, der Wabe oder Mittelwand hält und sich zur Durchsicht herausheben lässt. |
| **Bienenstock (Hive)** | Die Beute und ihr Volk. In Openbeehive ist ein Bienenstock die Einheit, die du durchsiehst und beschriftest. Siehe [Bienenstöcke](/using-the-app/hives) und [Beutentypen](/knowledge-base/hive-types). |
| **Stockmeißel (Hive tool)** | Ein flacher Metallhebel zum Auseinanderhebeln von Zargen und mit Propolis verklebten Rähmchen. |
| **Durchsicht (Inspection)** | Ein Besuch, bei dem du einen Bienenstock öffnest, seinen Zustand prüfst und festhältst, was du vorfindest. Siehe [Durchsicht](/beekeeping/inspecting). |
| **Eierlegende Arbeiterin (Laying worker)** | Eine Arbeiterin, die anfängt, (ausschließlich Drohnen-)Eier zu legen, wenn ein Volk hoffnungslos weisellos ist. |
| **Nektartracht (Nectar flow)** | Ein Zeitraum, in dem Pflanzen reichlich Nektar liefern und Völker rasch an Gewicht zulegen; auch „Tracht“ genannt. |
| **Ableger (Nucleus, Nuc)** | Ein kleines Anfangsvolk - typischerweise 3-5 Waben mit einer Königin, Brut, Futter und Bienen. Eine gängige Art, Bienen zu kaufen oder Völker zu vermehren. |
| **Propolis** | Ein klebriges Pflanzenharz, das Bienen verwenden, um Spalten abzudichten und Oberflächen zu überziehen. |
| **Königin (Queen)** | Das einzige fruchtbare Weibchen, das die Eier legt. Ihre Qualität bestimmt das gesamte Volk. Siehe [Königinnen](/using-the-app/queens). |
| **Weiselzelle (Queen cell)** | Eine große, erdnussförmige Zelle, in der eine neue Königin herangezogen wird. Ihr Vorhandensein deutet auf Schwarmtrieb, stille Umweiselung oder Notumweiselung hin. |
| **Absperrgitter (Queen excluder)** | Ein Gitter, dessen Maße Arbeiterinnen passieren lassen, die größere Königin aber nicht; es hält die Brut aus den Honigräumen heraus. |
| **Regentschaft (Reign)** | Der Zeitraum, in dem eine bestimmte Königin ein Volk führt, von der Zusetzung oder dem Schlupf bis zu ihrem Ersatz oder Verlust. Openbeehive verfolgt dies, damit die Volkshistorie über Königinnen hinweg durchgängig bleibt. |
| **Umweiselung (Requeening)** | Der Austausch der Königin eines Volkes, sei es zur Verbesserung der Sanftmut, der Leistung oder einfach, weil sie versagt. |
| **Honigraum (Super)** | Eine Zarge, die über dem Brutraum aufgesetzt wird, meist über einem Absperrgitter, in der die Bienen überschüssigen Honig zur Ernte einlagern. |
| **Stille Umweiselung (Supersedure)** | Das stille Heranziehen einer neuen Königin durch das Volk, um eine versagende zu ersetzen, meist ohne zu schwärmen. |
| **Schwarm (Swarm)** | Die Vermehrung des Volkes: Die alte Königin verlässt mit etwa der Hälfte der Bienen den Stock, um eine neue Behausung zu gründen. Siehe [Schwärmen](/beekeeping/swarming). |
| **Behandlung (Treatment)** | Ein Eingriff gegen Schädlinge oder Krankheiten, meist gegen Varroa. Siehe [Behandlungen](/using-the-app/treatments) und [Varroa](/beekeeping/varroa). |
| **Varroa** | *Varroa destructor*, eine parasitäre Milbe, die Bienen schwächt und Viren überträgt; in den meisten Regionen der schwerwiegendste Schädling. Siehe [Varroa](/beekeeping/varroa). |
| **Wartezeit (Withdrawal period)** | Die Zeit, die nach einer Behandlung vergehen muss, bevor Honig zum menschlichen Verzehr geerntet werden darf. Sie variiert je nach Produkt und Land - befolge stets die Packungsangaben. |
| **Arbeiterin (Worker)** | Eine unfruchtbare weibliche Biene. Arbeiterinnen verrichten nahezu die gesamte Arbeit des Volkes: Pflege, Bau, Bewachung und Sammeln. |

:::caution
Wartezeiten sind eine rechtliche und lebensmittelhygienische Angelegenheit, und
sie unterscheiden sich je nach Produkt und Land. Betrachte die Packungsangabe als
maßgeblich und prüfe deine nationalen Vorschriften.
:::

## Openbeehive-Begriffe

| Begriff | Bedeutung |
| --- | --- |
| **Event** | Ein nur anfügbarer Eintrag über etwas, das geschehen ist (zum Beispiel eine Ernte oder eine Behandlung). Events kollidieren während der Synchronisierung nie. Siehe [Historie und Events](/developers/history-and-events). |
| **HLC (Hybrid Logical Clock)** | Das Zeitstempelverfahren, das Openbeehive verwendet, um auf verschiedenen Geräten vorgenommene Änderungen zu ordnen, selbst offline, damit die Synchronisierung konsistent bleibt. Siehe [Sync-Protokoll](/developers/sync-protocol). |
| **Markierungsfarbe** | Die internationale Königinnen-Markierungsfarbe für das Jahr, in dem sie zugesetzt wurde (weiß, gelb, rot, grün, blau in einem Fünfjahreszyklus). Siehe [Königinnen-Markierungsfarben](/knowledge-base/queen-marking-colours). |
| **Offline-first** | Das Designprinzip, dass alles lokal und sofort funktioniert, ohne Empfang, und im Hintergrund synchronisiert wird, sobald eine Verbindung zurückkehrt. Siehe [Offline und Synchronisierung](/using-the-app/offline-and-sync). |
| **OPFS** | Das Origin Private File System, ein privater Bereich in deinem Browser, in dem Openbeehive seine lokale SQLite-Datenbank speichert. |
| **OR-Set** | Eine „Add-wins“-Menge, die für Listenfelder verwendet wird, sodass gleichzeitige Hinzufügungen von verschiedenen Geräten allesamt eine Zusammenführung überstehen. Siehe [Sync-Protokoll](/developers/sync-protocol). |
| **PWA** | Progressive Web App - die installierbare, offlinefähige Web-App, die du auf Smartphone, Tablet oder Desktop ausführst. Siehe [Installation](/using-the-app/install). |
| **QR-Etikett** | Ein druckbarer Code an einem Bienenstock, der einen Deep Link enthält; durch das Scannen öffnet sich die App direkt bei diesem Bienenstock. Siehe [QR-Etiketten](/using-the-app/qr-labels). |
| **Scope** | Die Freigabegrenze in Openbeehive. Das Teilen erfolgt auf Ebene des Bienenstands: Ein Scope gewährt anderen Zugriff auf einen Bienenstand und alles darin. |
| **Synchronisierung (Sync)** | Das konfliktfreie Zusammenführen deiner lokalen Änderungen mit dem Server und deinen anderen Geräten im Hintergrund. Siehe [Offline und Synchronisierung](/using-the-app/offline-and-sync). |
| **Aufgabe (Task)** | Eine Erinnerung oder ein To-do, das an einen Bienenstock oder Bienenstand gebunden ist - zum Beispiel „Honigraum aufsetzen“ oder „gegen Varroa behandeln“. Siehe [Aufgaben](/using-the-app/tasks). |

:::note
Die tiefergehende Mechanik von HLCs, OR-Sets und nur anfügbaren Events wird im
[Entwicklerbereich](/category/developers) behandelt. Für den täglichen Gebrauch
musst du nur wissen, dass deine Daten geräteübergreifend sauber zusammengeführt
werden, ohne dich je um die Lösung eines Konflikts zu bitten.
:::
