# Glossario

L'apicoltura ha un vocabolario tutto suo, e Openbeehive vi aggiunge qualche
termine proprio. Questa pagina raccoglie le parole che incontrerai più spesso,
sia in apiario sia nell'app, con definizioni brevi e pratiche.

Usa la funzione di ricerca del browser per saltare direttamente a un termine,
oppure scorri semplicemente la lista. Quando un termine ha una sua pagina
dedicata, vi rimandiamo con un link.

:::tip
Sei alle prime armi con l'apicoltura? Inizia da [la colonia](/beekeeping/the-colony) e
da [primi passi](/beekeeping/getting-started), poi torna qui ogni volta che una parola ti mette
in difficoltà.
:::

## Termini di apicoltura

| Termine | Significato |
| --- | --- |
| **Apiario** | Un luogo in cui si tengono uno o più alveari; un "apiario". In Openbeehive l'apiario è il vertice della gerarchia e l'unità di condivisione. Vedi [apiari](/using-the-app/apiaries). |
| **Spazio d'ape** | Lo spazio di circa 6-9 mm che le api lasciano come passaggio. Gli spazi più piccoli vengono riempiti con propoli; quelli più grandi con favo. Uno spazio d'ape corretto mantiene i telaini mobili. |
| **Covata** | I giovani in sviluppo: uova, larve e pupe. Una "configurazione della covata" sana è compatta e con poche lacune. |
| **Nido** | La cassetta (o le cassette) inferiore in cui la regina depone e si alleva la covata, in contrapposizione ai melari soprastanti. |
| **Favo irregolare** | Favo costruito a caso negli spazi vuoti o tra le cassette, al di fuori dei telaini. |
| **Covata opercolata** | Pupe sigillate sotto gli opercoli di cera. Gli opercoli delle operaie sono piatti e opachi; quelli dei fuchi sono bombati. |
| **Cella** | Un singolo scomparto esagonale del favo. Tra le celle speciali ci sono le celle reali, le celle da fuco e le celle da fondazione. |
| **Favo** | La struttura di cera fatta di celle esagonali usata per immagazzinare miele e polline e per allevare la covata. |
| **Favo costruito** | Foglio cereo che le api hanno elaborato fino a formare celle complete e utilizzabili. |
| **Fuco** | Un'ape maschio. I fuchi non bottinano e non hanno pungiglione; il loro ruolo è accoppiarsi con le regine. |
| **Regina fucaiola** | Una regina in declino o non fecondata che depone solo uova non fecondate (da fuco) - segno che va sostituita. |
| **Foglio cereo** | Un foglio di cera d'api o di plastica, spesso goffrato con il disegno delle celle, che offre alle api una base su cui costruire il favo. |
| **Bottinatura** | Il nettare, il polline, l'acqua e la propoli che le api raccolgono; anche l'atto di raccoglierli. |
| **Telaino** | Un rettangolo rimovibile di legno o plastica che sostiene il favo o il foglio cereo e permette di estrarlo per l'ispezione. |
| **Alveare** | La cassetta e la sua colonia. In Openbeehive l'alveare è l'unità che ispezioni ed etichetti. Vedi [alveari](/using-the-app/hives) e [tipi di alveare](/knowledge-base/hive-types). |
| **Leva da apicoltore** | Una leva piatta di metallo per separare cassette e telaini incollati dalla propoli. |
| **Ispezione** | Una visita durante la quale apri un alveare, ne verifichi le condizioni e annoti ciò che trovi. Vedi [ispezionare](/beekeeping/inspecting). |
| **Operaia ovificatrice** | Un'operaia che inizia a deporre uova (solo da fuco) quando una colonia è irrimediabilmente orfana. |
| **Importazione** | Un periodo in cui le piante producono nettare in abbondanza e le colonie aumentano rapidamente di peso; chiamato anche "flusso". |
| **Nucleo** | Una piccola colonia di avviamento - di solito 3-5 telaini con una regina, covata, scorte e api. Un modo comune per acquistare api o fare aumento. |
| **Propoli** | Una resina vegetale appiccicosa che le api usano per sigillare le fessure e verniciare le superfici. |
| **Regina** | L'unica femmina fertile che depone le uova. La sua qualità guida l'intera colonia. Vedi [regine](/using-the-app/queens). |
| **Cella reale** | Una cella grande, a forma di arachide, in cui viene allevata una nuova regina. La loro presenza segnala sciamatura, sostituzione o orfanità d'emergenza. |
| **Escludi regina** | Una griglia dimensionata in modo che le operaie passino ma la regina, più grande, no, usata per tenere la covata fuori dai melari. |
| **Regno** | Il periodo in cui una determinata regina è a capo di una colonia, dall'introduzione o dalla nascita fino a quando viene sostituita o persa. Openbeehive lo tiene traccia così che la storia della colonia resti continua attraverso le diverse regine. |
| **Sostituzione della regina** | La sostituzione della regina di una colonia, sia per migliorarne il carattere o la produttività, sia semplicemente perché è in declino. |
| **Melario** | Una cassetta aggiunta sopra il nido, di solito sopra un escludi regina, in cui le api immagazzinano il miele in eccesso per la raccolta. |
| **Sostituzione naturale (supersedure)** | La colonia che alleva silenziosamente una nuova regina per sostituire quella in declino, di solito senza sciamare. |
| **Sciame** | La riproduzione della colonia: la vecchia regina parte con circa metà delle api per fondare una nuova dimora. Vedi [sciamatura](/beekeeping/swarming). |
| **Trattamento** | Un intervento contro parassiti o malattie, più spesso contro la varroa. Vedi [trattamenti](/using-the-app/treatments) e [varroa](/beekeeping/varroa). |
| **Varroa** | *Varroa destructor*, un acaro parassita che indebolisce le api e diffonde virus; il parassita più grave nella maggior parte delle regioni. Vedi [varroa](/beekeeping/varroa). |
| **Tempo di sospensione** | Il tempo che deve trascorrere dopo un trattamento prima che il miele possa essere raccolto per il consumo umano. Varia in base al prodotto e al paese - segui sempre l'etichetta. |
| **Operaia** | Un'ape femmina infertile. Le operaie svolgono quasi tutto il lavoro della colonia: accudire, costruire, fare la guardia e bottinare. |

:::caution
I tempi di sospensione sono una questione legale e di sicurezza alimentare, e
differiscono tra i prodotti e tra i paesi. Considera l'etichetta come l'autorità
di riferimento e verifica le norme nazionali.
:::

## Termini di Openbeehive

| Termine | Significato |
| --- | --- |
| **Evento** | Un record di sola aggiunta di qualcosa che è accaduto (per esempio una raccolta o un trattamento). Gli eventi non entrano mai in conflitto durante la sincronizzazione. Vedi [storia ed eventi](/developers/history-and-events). |
| **HLC (Hybrid Logical Clock)** | Lo schema di marcatura temporale che Openbeehive usa per ordinare le modifiche fatte su dispositivi diversi, anche offline, così da mantenere coerente la sincronizzazione. Vedi [protocollo di sincronizzazione](/developers/sync-protocol). |
| **Colore di marcatura** | Il colore internazionale di marcatura della regina per l'anno in cui è stata introdotta (bianco, giallo, rosso, verde, blu su un ciclo quinquennale). Vedi [colori di marcatura delle regine](/knowledge-base/queen-marking-colours). |
| **Offline-first** | Il principio di progettazione per cui tutto funziona localmente e all'istante, senza segnale, e si sincronizza in background quando torna la connessione. Vedi [offline e sincronizzazione](/using-the-app/offline-and-sync). |
| **OPFS** | L'Origin Private File System, un'area privata del browser in cui Openbeehive memorizza il suo database SQLite locale. |
| **OR-Set** | Un insieme "add-wins" usato per i campi a lista, così che le aggiunte concorrenti da dispositivi diversi sopravvivano tutte a una fusione. Vedi [protocollo di sincronizzazione](/developers/sync-protocol). |
| **PWA** | Progressive Web App - l'app web installabile e utilizzabile offline che esegui su telefono, tablet o desktop. Vedi [installazione](/using-the-app/install). |
| **Etichetta QR** | Un codice stampabile su un alveare che codifica un deep link; scansionandolo si apre l'app direttamente su quell'alveare. Vedi [etichette QR](/using-the-app/qr-labels). |
| **Scope (ambito)** | Il confine di condivisione in Openbeehive. La condivisione avviene a livello di apiario: uno scope concede ad altri l'accesso a un apiario e a tutto ciò che contiene. |
| **Sincronizzazione** | La fusione in background, senza conflitti, delle tue modifiche locali con il server e i tuoi altri dispositivi. Vedi [offline e sincronizzazione](/using-the-app/offline-and-sync). |
| **Attività** | Un promemoria o una cosa da fare legata a un alveare o a un apiario - per esempio "aggiungere un melario" o "trattare contro la varroa". Vedi [attività](/using-the-app/tasks). |

:::note
I meccanismi più approfonditi di HLC, OR-Set ed eventi di sola aggiunta sono
trattati nella [sezione per sviluppatori](/category/developers). Per l'uso quotidiano ti basta sapere che
i tuoi dati si fondono in modo pulito tra i dispositivi senza mai chiederti di
risolvere un conflitto.
:::
