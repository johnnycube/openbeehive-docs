# Glosario

La apicultura tiene un vocabulario propio, y Openbeehive añade algunos términos
adicionales por encima. Esta página reúne las palabras que encontrarás con más
frecuencia, tanto en la colmena como en la aplicación, con definiciones breves y
prácticas.

Usa la función de búsqueda de tu navegador para saltar directamente a un término,
o simplemente échale un vistazo. Cuando un término tiene su propia página, la
enlazamos.

:::tip
¿Nuevo en la apicultura? Empieza por [la colonia](/beekeeping/the-colony) y
[primeros pasos](/beekeeping/getting-started), y vuelve aquí siempre que una
palabra te haga dudar.
:::

## Términos de apicultura

| Término | Significado |
| --- | --- |
| **Colmenar (apiario)** | Un emplazamiento donde se mantienen una o más colmenas. En Openbeehive un colmenar es la cima de la jerarquía y la unidad de compartición. Consulta [colmenares](/using-the-app/apiaries). |
| **Espacio de abeja** | El hueco de aproximadamente 6-9 mm que las abejas dejan como paso. Los espacios menores se rellenan con propóleo; los mayores se rellenan con panal. Un espacio de abeja correcto mantiene los cuadros movibles. |
| **Cría** | Las jóvenes en desarrollo: huevos, larvas y pupas. Un buen "patrón de cría" es compacto y con pocos huecos. |
| **Cámara de cría** | La caja inferior (o cajas) donde la reina pone y se cría la prole, a diferencia de las alzas que están encima. |
| **Panal irregular (burr comb)** | Panal disperso construido en huecos o entre cajas, fuera de los cuadros. |
| **Cría opercula** | Pupas selladas bajo opérculos de cera. Los opérculos de obrera son planos y mates; los de zángano son abombados. |
| **Celda (celdilla)** | Un único compartimento hexagonal del panal. Entre las celdas especiales están las celdas reales, las celdas de zángano y las cazoletas. |
| **Panal** | La estructura de cera de celdas hexagonales usada para almacenar miel, polen y criar prole. |
| **Panal estirado** | Cera estampada (lámina) que las abejas han construido hasta formar celdas completas y utilizables. |
| **Zángano** | Una abeja macho. Los zánganos no pecorean y no tienen aguijón; su función es aparearse con las reinas. |
| **Reina zanganera** | Una reina defectuosa o no fecundada que solo pone huevos no fertilizados (de zángano): señal de que necesita ser reemplazada. |
| **Lámina (cera estampada)** | Una hoja de cera de abeja o plástico, a menudo grabada con un patrón de celdas, que da a las abejas una base sobre la que estirar el panal. |
| **Pecorea (forrajeo)** | El néctar, polen, agua y propóleo que recolectan las abejas; también el acto de recolectarlo. |
| **Cuadro** | Un rectángulo extraíble de madera o plástico que sostiene panal o lámina y permite levantarlo para la inspección. |
| **Colmena** | La caja y su colonia. En Openbeehive una colmena es la unidad que inspeccionas y etiquetas. Consulta [colmenas](/using-the-app/hives) y [tipos de colmena](/knowledge-base/hive-types). |
| **Palanca (hive tool)** | Una palanca metálica plana para separar cajas y cuadros pegados con propóleo. |
| **Inspección** | Una visita durante la cual abres una colmena, compruebas su estado y registras lo que encuentras. Consulta [inspección](/beekeeping/inspecting). |
| **Obrera ponedora** | Una obrera que empieza a poner huevos (solo de zángano) cuando una colonia se queda irremediablemente huérfana. |
| **Flujo de néctar** | Un periodo en que las plantas producen néctar abundantemente y las colonias ganan peso rápidamente; también llamado "entrada de néctar". |
| **Núcleo (nuc)** | Una pequeña colonia inicial, normalmente de 3-5 cuadros con una reina, cría, alimento y abejas. Una forma habitual de comprar abejas o multiplicar. |
| **Propóleo** | Una resina vegetal pegajosa que las abejas usan para sellar huecos y barnizar superficies. |
| **Reina** | La única hembra fértil que pone los huevos. Su calidad determina toda la colonia. Consulta [reinas](/using-the-app/queens). |
| **Celda real** | Una celda grande, con forma de cacahuete, en la que se cría una nueva reina. Su presencia indica enjambrazón, reemplazo (supersedura) o reposición de emergencia. |
| **Excluidor de reinas** | Una rejilla dimensionada para que pasen las obreras pero no la reina, más grande, usada para mantener la cría fuera de las alzas de miel. |
| **Reinado** | El periodo en que una reina concreta encabeza una colonia, desde su introducción o emergencia hasta que es reemplazada o se pierde. Openbeehive lo registra para que el historial de la colonia siga siendo continuo entre reinas. |
| **Reposición de reina (requeening)** | Reemplazar la reina de una colonia, ya sea para mejorar el temperamento, la productividad o simplemente porque está fallando. |
| **Alza** | Una caja añadida por encima de la cría, normalmente sobre un excluidor de reinas, en la que las abejas almacenan miel excedente para la cosecha. |
| **Supersedura (reemplazo)** | Cuando la colonia cría tranquilamente una nueva reina para reemplazar a una que falla, normalmente sin enjambrar. |
| **Enjambre (enjambrazón)** | La reproducción de la colonia: la reina vieja se marcha con cerca de la mitad de las abejas para fundar un nuevo hogar. Consulta [enjambrazón](/beekeeping/swarming). |
| **Tratamiento** | Una intervención contra plagas o enfermedades, casi siempre contra la varroa. Consulta [tratamientos](/using-the-app/treatments) y [varroa](/beekeeping/varroa). |
| **Varroa** | *Varroa destructor*, un ácaro parásito que debilita a las abejas y propaga virus; la plaga más grave en la mayoría de las regiones. Consulta [varroa](/beekeeping/varroa). |
| **Periodo de carencia** | El tiempo que debe transcurrir tras un tratamiento antes de poder cosechar miel para consumo humano. Varía según el producto y el país: sigue siempre la etiqueta. |
| **Obrera** | Una abeja hembra infértil. Las obreras realizan casi todo el trabajo de la colonia: cuidado de la cría, construcción, vigilancia y pecorea. |

:::caution
Los periodos de carencia son una cuestión legal y de seguridad alimentaria, y
difieren entre productos y entre países. Considera la etiqueta como la autoridad
y consulta las normas de tu país.
:::

## Términos de Openbeehive

| Término | Significado |
| --- | --- |
| **Evento** | Un registro de solo adición de algo que ocurrió (por ejemplo una cosecha o un tratamiento). Los eventos nunca entran en conflicto durante la sincronización. Consulta [historial y eventos](/developers/history-and-events). |
| **HLC (reloj lógico híbrido)** | El esquema de marcas de tiempo que Openbeehive usa para ordenar los cambios hechos en distintos dispositivos, incluso sin conexión, de modo que la sincronización se mantenga coherente. Consulta [protocolo de sincronización](/developers/sync-protocol). |
| **Color de marcaje** | El color internacional de marcaje de reinas correspondiente al año en que fue introducida (blanco, amarillo, rojo, verde, azul en un ciclo de cinco años). Consulta [colores de marcaje de reinas](/knowledge-base/queen-marking-colours). |
| **Offline-first** | El principio de diseño según el cual todo funciona localmente y de forma instantánea, sin señal, y se sincroniza en segundo plano cuando vuelve la conexión. Consulta [sin conexión y sincronización](/using-the-app/offline-and-sync). |
| **OPFS** | El Origin Private File System, un área privada de tu navegador donde Openbeehive almacena su base de datos SQLite local. |
| **OR-Set** | Un conjunto de tipo "gana la adición" usado para campos de lista, de modo que las adiciones concurrentes desde distintos dispositivos sobrevivan todas a una fusión. Consulta [protocolo de sincronización](/developers/sync-protocol). |
| **PWA** | Progressive Web App: la aplicación web instalable y capaz de funcionar sin conexión que ejecutas en el teléfono, la tableta o el escritorio. Consulta [instalación](/using-the-app/install). |
| **Etiqueta QR** | Un código imprimible en una colmena que codifica un enlace profundo; al escanearlo se abre la aplicación directamente en esa colmena. Consulta [etiquetas QR](/using-the-app/qr-labels). |
| **Ámbito (scope)** | El límite de compartición en Openbeehive. La compartición se produce a nivel de colmenar: un ámbito concede a otros acceso a un colmenar y a todo lo que contiene. |
| **Sincronización** | La fusión en segundo plano y sin conflictos de tus cambios locales con el servidor y tus otros dispositivos. Consulta [sin conexión y sincronización](/using-the-app/offline-and-sync). |
| **Tarea** | Un recordatorio o pendiente ligado a una colmena o colmenar; por ejemplo "añadir un alza" o "tratar contra varroa". Consulta [tareas](/using-the-app/tasks). |

:::note
Los mecanismos más profundos de los HLC, los OR-Sets y los eventos de solo
adición se tratan en la [sección para desarrolladores](/category/developers).
Para el uso diario solo necesitas saber que tus datos se fusionan limpiamente
entre dispositivos sin pedirte nunca que resuelvas un conflicto.
:::
